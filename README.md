# To Do app

This is my implementation of the backend for the To Do app.

I have used FastAPI with python.

# Requirements

- Python 3.10
- Docker Desktop 4.20
- Install FastAPI and uvicorn modules

# How to run

```
$Env:DB_HOST="localhost"
$Env:DB_DATABASE="db"
$Env:DB_USER="username"
$Env:DB_PASSWORD="example"
```
